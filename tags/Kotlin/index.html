Kotlin：内置排序方法

Kotlin中内置的数组排序的方法有sort, sorted, sortBy, sortedBy, sortWith, sortedWith。可以粗略地对这些方法进行分类。

#Kotlin
Kotlin：构造器和初始化

Kotlin 中有两种构造器：主构造器和次构造器。都用关键字 constructor 来表示。但是在每个类中，主构造器有且仅有一个（如果没有定义，则编译器会自动提供一个空构造器），而次构造器可以有 0 个或多个。
